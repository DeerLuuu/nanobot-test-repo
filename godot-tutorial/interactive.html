GDScript 基础教程

什么是 GDScript？

GDScript 是 Godot 引擎专用的脚本语言，语法类似 Python，但针对游戏开发进行了优化。 它与 Godot 引擎深度集成，可以轻松访问节点、资源和场景。

变量声明

使用 var 关键字声明变量，支持类型推断和显式类型。

var name = "玩家" # 类型推断
var score: int = 100 # 显式类型

数据类型

GDScript 支持多种内置数据类型，包括基础类型和 Godot 特有类型。

int, float, bool, String
Array, Dictionary
Vector2, Vector3, Color

print() 函数

用于在控制台输出调试信息，支持字符串拼接和格式化。

print("Hello Godot!")
print("分数: " + str(100))
print("x=%d, y=%d" % [10, 20])

类型转换

使用内置函数在不同类型之间转换。

str(100) # 整数转字符串
int("50") # 字符串转整数
float("3.14") # 字符串转浮点
