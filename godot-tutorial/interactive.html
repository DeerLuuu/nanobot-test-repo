变量声明

在 GDScript 中，使用 var 关键字声明变量。GDScript 支持类型推断，也支持显式类型声明。

类型推断

让 GDScript 自动推断变量类型

var name = "玩家"
var score = 100
var speed = 5.5

显式类型

明确指定变量类型

var name: String = "玩家"
var score: int = 100
var speed: float = 5.5

数据类型

GDScript 支持多种内置数据类型：

基础类型

数字、布尔值、字符串

int      # 整数: 1, 100, -50
float    # 浮点: 3.14, -0.5
bool     # 布尔: true, false
String   # 字符串: "Hello"

Godot 类型

Godot 引擎特有类型

Vector2  # 2D向量
Vector3  # 3D向量
Color    # 颜色
Array    # 数组
Dictionary # 字典

print() 函数

print() 是最常用的调试函数，用于在控制台输出信息。

基本用法

输出字符串或变量

print("Hello Godot!")
print(player_name)
print("分数: " + str(score))

多参数

输出多个值（用逗号分隔）

print("名称:", name)
print("x:", x, "y:", y)
print("HP:", hp, "/", max_hp)

类型转换

使用内置函数在不同类型之间转换：

转字符串

将其他类型转为字符串

str(100)      # "100"
str(3.14)     # "3.14"
str(true)     # "true"

转数字

将字符串转为数字

int("100")    # 100
float("3.14") # 3.14
int(3.9)      # 3 (截断)
