Godot print() 函数教程

print() 函数是Godot中用于调试和输出信息的基本函数。它可以将文本、变量和表达式的结果输出到控制台。

基本用法:

  • print("Hello World") - 打印字符串
  • print(variable_name) - 打印变量值
  • print("Value: " + str(number)) - 拼接字符串和数字

在代码编辑器中尝试以下示例:

基础打印
print("欢迎学习Godot!")
变量打印
var x = 10; print("x = " + str(x))
数学运算
print("5 + 3 = " + str(5 + 3))
条件语句
if true: print("条件为真")
