    2 安装 Godot 引擎

    方式1: 从官网下载（推荐）

    • 访问 Godot官网下载页面
    • 选择 Godot 4.x 版本（建议使用最新稳定版）
    • 下载 Standard version（标准版）即可
    • 解压到合适位置（如 C:\Godot\）
    • 双击 Godot_v4.x.x_stable_win64.exe 启动

    模拟下载流程

    点击下方按钮模拟Godot下载过程：

    下载完成！ 请解压后运行 Godot_v4.x.x_stable_win64.exe

    方式2: 通过Steam安装

    • 打开 Steam 客户端
    • 在商店中搜索 "Godot"
    • 点击"安装"按钮
    • 安装完成后在库中启动
    选择建议
    • 官网下载: 更灵活，适合需要多版本切换的开发者
    • Steam安装: 自动更新，适合Steam用户

    3 安装 Git 和版本控制工具

    安装步骤 终端命令

    4 Git 基础命令详解

    掌握以下核心命令，即可应对日常开发需求：

    git-workflow.sh 可交互
    # === 仓库操作 === # 克隆远程仓库到本地 git clone https://github.com/用户名/项目名.git # 初始化新仓库 git init # === 日常操作 === # 查看当前状态 git status # 添加所有更改到暂存区 git add . # 添加特定文件 git add 文件名 # 提交更改（附带说明） git commit -m "描述本次更改的内容" # 推送到远程仓库 git push origin main # 从远程拉取更新 git pull origin main # === 分支操作 === # 创建并切换到新分支 git checkout -b feature/新功能名 # 查看所有分支 git branch # 切换分支 git checkout main # 合并分支 git merge feature/新功能名
    Git 工作流程图解

    工作区 → git add → 暂存区 → git commit → 本地仓库 → git push → 远程仓库

    5 创建 Godot 项目

    项目创建步骤

    1. 打开 Godot 引擎
    2. 点击右侧的 "新建" 按钮
    3. 设置项目名称：godot-hd2d-survival
    4. 选择项目保存路径（建议使用英文路径）
    5. 渲染器选择：Forward+（推荐，画质好）
    6. 点击 "创建并编辑"
    注意事项

    • 避免使用中文路径，可能导致资源加载问题
    • 推荐的项目结构可以帮助你更好地组织代码

    推荐的项目文件夹结构

    godot-hd2d-survival/ ├── project.godot # 项目配置文件 ├── icon.svg # 项目图标 │ ├── scenes/ # 场景文件目录 │ ├── player/ # 玩家相关场景 │ ├── enemies/ # 敌人场景 │ ├── world/ # 世界场景 │ └── ui/ # UI场景 │ ├── scripts/ # GDScript脚本目录 │ ├── player/ # 玩家脚本 │ ├── enemies/ # 敌人AI脚本 │ └── utils/ # 工具类脚本 │ ├── assets/ # 资源文件目录 │ ├── sprites/ # 精灵图片 │ ├── audio/ # 音频文件 │ ├── fonts/ # 字体文件 │ └── animations/ # 动画资源 │ ├── resources/ # Godot资源文件 │ └── themes/ # UI主题资源 │ └── addons/ # 插件目录（可选）

    交互学习区

    问题1: Godot是什么类型的软件？
    • A. 游戏引擎
    • B. 图像编辑器
    • C. 视频播放器
    • D. 音乐制作软件
    解析：Godot是一款免费开源的游戏引擎，支持2D和3D游戏开发，使用GDScript作为主要脚本语言。
    问题2: Git的主要作用是什么？
    • A. 编辑代码
    • B. 版本控制
    • C. 运行程序
    • D. 设计界面
    解析：Git是一个分布式版本控制系统，用于跟踪代码变更、协同开发和项目管理。
    问题3: GDScript是哪种语言的变体？
    • A. Java
    • B. C++
    • C. Python
    • D. JavaScript
    解析：GDScript的语法设计参考了Python，但针对Godot引擎进行了优化，具有更好的游戏开发支持。
