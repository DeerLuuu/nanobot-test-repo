    • 2 安装 Godot

    方式1: 从官网下载

    访问 Godot官网 下载最新版本

    下载完成！解压到合适位置后运行 Godot_v4.6.1-stable_win64.exe

    方式2: 通过Steam安装

    在Steam商店中搜索并安装Godot

    3 安装 Git 和 SourceTree

    安装步骤 命令行
    # 1. 安装 Git # 访问: https://git-scm.cn/ # 点击 Download 按钮 # 2. 安装 SourceTree # 访问: https://sourcetreeapp.com/ # 下载对应平台的版本 # 3. 配置用户信息 git config --global user.name "你的名字" git config --global user.email "你的邮箱"

    4 Git 基础使用

    基础Git命令 可交互
    # 克隆仓库到本地 git clone https://github.com/你的用户名/godot-hd2d-survival.git # 进入项目目录 cd godot-hd2d-survival # 创建新分支 git checkout -b feature/initial-setup # 添加文件到暂存区 git add . # 提交更改 git commit -m "初始项目设置" # 推送到远程仓库 git push origin feature/initial-setup
