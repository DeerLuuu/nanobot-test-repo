# Godot HD2D生存游戏交互式教程

## 🎮 项目介绍

这是一个基于Godot 4的HD2D生存游戏交互式教程网站。通过这个教程，你可以从零开始学习如何使用Godot引擎开发一个类似《饥荒》的2.5D生存游戏。

## 🌐 在线访问

访问地址: [https://deerluuu.github.io/nanobot-test-repo/godot-tutorial/](https://deerluuu.github.io/nanobot-test-repo/godot-tutorial/)

## 📚 教程特点

### 1. 交互式学习体验
- **实时代码编辑器**: 在浏览器中直接运行和修改代码
- **命令行模拟器**: 学习Git命令的交互式终端
- **进度跟踪系统**: 可视化学习进度和完成情况
- **即时反馈**: 答案检查和错误提示

### 2. 完整课程体系
- **8周学习计划**: 从基础到高级的完整路径
- **14个详细章节**: 覆盖游戏开发所有核心概念
- **实战项目驱动**: 通过实际项目学习技术

### 3. 多维度学习资源
- **视频教程**: 配合文字教程的视频讲解
- **代码示例**: 可直接运行的完整代码
- **检查清单**: 确保每个步骤都正确完成
- **项目模板**: 快速开始新项目的模板

## 🚀 快速开始

### 1. 本地运行
```bash
# 克隆仓库
git clone https://github.com/DeerLuuu/nanobot-test-repo.git

# 进入教程目录
cd nanobot-test-repo/godot-tutorial

# 使用本地服务器运行
# 方法1: 使用Python
python -m http.server 8000

# 方法2: 使用Node.js
npx serve .

# 然后在浏览器中访问
# http://localhost:8000
```
