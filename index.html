如何请求网页项目

当你需要创建网页时，只需告诉 nanobot：

  1. 描述功能 - 详细描述你想要的网页功能
  2. 说明风格 - 说明设计风格偏好（如果有）
  3. 指定功能 - 指定需要的特殊功能或 API 集成
  4. 等待创建 - nanobot 会创建网页并提交到仓库
  5. 获取链接 - 你会收到网页的访问链接
