Slidev演示

在静态网页项目中使用Slidev创建交互式演示文稿

Slidev是一个基于Web的幻灯片制作工具，专为开发者设计。它使用Markdown编写内容，支持Vue组件、代码高亮和交互式演示。

快速开始

使用Markdown快速创建专业的演示文稿，无需复杂的图形界面。

代码友好

支持代码高亮、实时执行和语法检查，非常适合技术演示。

静态部署

构建为静态单页应用，支持GitHub Pages、Netlify等静态托管服务。

查看演示文稿 Slidev官网 查看Markdown源文件

Slidev主要特性

Markdown编写

使用熟悉的Markdown语法编写幻灯片内容

主题系统

丰富的主题和自定义选项

热重载

编辑时实时预览，快速迭代

Vue组件

支持Vue组件，创建交互式演示

响应式设计

适配各种屏幕尺寸和设备

多种导出

支持PDF、PNG、PPTX等多种格式

构建命令

# 安装依赖
npm install

# 开发模式（热重载）
npm run dev

# 构建静态页面（用于部署）
npm run build --base /nanobot-test-repo/slidev-demo/

# 导出为PDF
npm run export

构建后的文件位于 dist/ 目录，可以直接部署到任何静态托管服务。
