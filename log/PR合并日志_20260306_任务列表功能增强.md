## 🔧 技术架构

### 1. 任务列表渲染流程
```
Markdown文档 → Marked.js解析 → 自定义渲染器 → HTML输出 → 样式应用 → 交互绑定
    ↓              ↓              ↓              ↓           ↓           ↓
- [ ] 任务   → 语法解析   → 检测任务列表 → <li class="task-list-item"> → 应用.task-list-item样式 → 绑定toggleTask事件
- [x] 任务   → 语法解析   → 检测已完成 → <input type="checkbox" checked> → 应用.task-list-item-checked样式 → 绑定toggleTask事件
```

### 2. 自定义渲染器实现
```javascript
// 重写 list 方法，检测是否包含任务列表
renderer.list = function(body, ordered, start) {
    const isTaskList = body.includes('class="task-list-item"');
    const listClass = isTaskList ? ' class="task-list"' : '';
    const tag = ordered ? 'ol' : 'ul';
    const startAttr = ordered && start !== 1 ? ` start="${start}"` : '';
    return `<${tag}${listClass}${startAttr}>${body}</${tag}>`;
};

// 重写 listitem 方法，处理任务列表
renderer.listitem = function(text, task, checked) {
    if (task) {
        const isChecked = checked ? 'checked' : '';
        const checkedClass = checked ? 'task-list-item-checked' : '';
        return `<li class="task-list-item ${checkedClass}">
            <input type="checkbox" class="task-checkbox" ${isChecked} onclick="toggleTask(this)">
            <span class="task-text">${text}</span>
        </li>`;
    }
    return `<li>${text}</li>`;
};
```

### 3. 交互功能实现
```javascript
// 任务列表交互
function toggleTask(checkbox) {
    const li = checkbox.closest('.task-list-item');
    if (li) {
        li.classList.toggle('task-list-item-checked', checkbox.checked);
    }
}
```
