## 📁 项目结构检查
### ✅ 目录结构
```
HD2DSurvival/
├── addons/          # 插件
├── assets/          # 资源
│   ├── audio/       # 音效音乐
│   ├── fonts/       # 字体
│   ├── icons/       # 图标
│   ├── models/      # 模型
│   └── img/         # 纹理
├── scenes/          # 场景
│   ├── ui/          # UI相关
├── shaders/         # 着色器
├── config/          # 配置文件
│   ├── item/        # 物品 Resource
│   ├── theme/       # 主题文件
├── scripts/         # 脚本
│   ├── autoload/    # 自动加载脚本
└── ├── utils/       # 工具类
```
### ✅ 文件命名规范
- [ ] 场景文件: `小写场景根节点名称.tscn` (如: `world.tscn`)
- [ ] 脚本文件: `小写节点名称.gd` (如: `player_controller.gd`)
- [ ] 资源文件: `类型_名称.tscn` (如: `item_wood.tscn`)
- [ ] 使用蛇形命名法 (snake_case)
### ✅ Git配置
- [ ] .gitignore文件配置正确
- [ ] README.md项目说明完整
- [ ] LICENSE许可证选择
- [ ] 远程仓库连接正常
